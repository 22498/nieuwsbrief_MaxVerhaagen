# nieuwsbrief_MaxVerhaagen - 22 Maart 2018

We moesten een online nieuwsbrief maken waarbij de bezoeker zich kan inschrijven voor de nieuwsbrief en ook weer uitschrijven als die dat wilt daar naast moest je ook een admin gedeelte te maken waarbij je een nieuwsbrief kan versturen naar alle mensen die zijn ingeschreven bij jou nieuwsbrief. Het laatste wat bij de admin gedeelte nog moest komen was een mail lijst waar jij kan zien wie er is ingeschreven en kan je ook mensen verwijderen uit de lijst die niet meer bij jou staan ingeschreven.
